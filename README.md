# qr-forge-light
QR-Forge Light ist ein lokaler Desktop-QR-Code-Generator für Windows. QR-Codes werden offline erstellt – ohne Cloud, ohne Datenübertragung. Die Demo eignet sich zur Evaluierung der Funktionen und des Workflows.  Funktionen:  Text, URL, Google Maps, Telefon, WhatsApp, SMS, vCard  Farben, Verläufe, Rahmen, Text &amp; Branding incl.
